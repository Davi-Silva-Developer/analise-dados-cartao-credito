# Análise de Dados de Cartão de Crédito com Python

Projeto acadêmico desenvolvido utilizando **Python** e **Pandas** para realizar uma análise exploratória de dados de clientes de cartão de crédito.

O objetivo do projeto é explorar um dataset contendo informações demográficas e financeiras de aproximadamente 30 mil clientes, identificando padrões e características dos dados.

## Objetivos da Análise

Durante o projeto foram realizadas as seguintes análises:

- Quantidade de linhas e colunas do dataset
- Identificação de variáveis categóricas e numéricas
- Resumo estatístico dos dados
- Frequência das características categóricas
- Verificação de dados faltantes
- Exploração inicial do dataset

## Tecnologias Utilizadas

- Python
- Pandas
- VS Code

## Sobre o Dataset

O dataset utilizado contém dados financeiros e demográficos de clientes de cartão de crédito, permitindo uma análise exploratória voltada à identificação de possíveis padrões relacionados à inadimplência.
